# プライバシーポリシー

最終更新日: 2026年1月25日

## はじめに

「傘持ってく？」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリがどのようにユーザーの情報を収集、使用、保護するかについて説明します。

## 収集する情報

### 位置情報
本アプリは、以下の目的で位置情報を収集することがあります：

- **おおよその位置情報**: ユーザーが現在いる都道府県を特定し、該当地域の天気予報を取得するために使用します
- **正確な位置情報**: GPS機能が有効な場合、より正確な現在地を特定するために使用します

**重要**: 位置情報は端末内でのみ処理され、外部サーバーに送信されることはありません。

### 収集しない情報
本アプリは以下の情報を収集しません：

- 氏名、メールアドレス、電話番号などの個人識別情報
- クレジットカード情報などの財務情報
- 健康情報
- 連絡先情報
- 閲覧履歴
- 使用状況データ（分析目的）

## 情報の使用目的

収集した位置情報は、以下の目的にのみ使用されます：

1. ユーザーの現在地に基づいた天気予報の取得
2. 傘の要否判断に必要な気象データの取得

## データの保存

本アプリが保存するデータは、すべてユーザーの端末内に保存されます：

- 登録した地点情報
- 通知設定
- 外出時間設定
- 傘判断基準の設定
- 天気データのキャッシュ（15分間のみ保持）

これらのデータは外部サーバーに送信されることはありません。

## 第三者サービス

本アプリは以下の第三者サービスを利用しています：

### 気象庁天気予報API
- **用途**: 天気予報データの取得
- **送信データ**: 都道府県コードのみ
- **プライバシーポリシー**: https://www.jma.go.jp/jma/kishou/info/coment.html

### OpenStreetMap Nominatim API
- **用途**: 住所・地名から座標への変換（地点検索機能）
- **送信データ**: ユーザーが入力した検索クエリ
- **プライバシーポリシー**: https://wiki.osmfoundation.org/wiki/Privacy_Policy

## データの共有

本アプリは、ユーザーの個人情報を第三者と共有、販売、または貸与することはありません。

## セキュリティ

本アプリは、ユーザーのデータを保護するために適切なセキュリティ対策を講じています。すべてのデータは端末内に保存され、外部への送信は最小限に抑えられています。

## お子様のプライバシー

本アプリは全年齢を対象としており、13歳未満のお子様から意図的に個人情報を収集することはありません。

## プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。変更があった場合は、本ページの最終更新日を更新し、重要な変更についてはアプリ内で通知します。
