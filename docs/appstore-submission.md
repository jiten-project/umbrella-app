## 説明文

### 短い説明 (プロモーションテキスト・170文字以内)
今日、傘がいるかどうかを一目で判断！気象庁の天気予報データを使って、外出時間帯の降水確率と降水量から傘の要否を自動判断します。出発地と目的地、両方の天気をチェックできます。

### 長い説明 (4000文字以内)
```
【傘持ってく？とは】
毎朝「今日、傘いるかな？」と悩む時間を解消！
気象庁の公式天気予報データを使って、あなたの外出時間帯に傘が必要かどうかを一目で判断できるアプリです。

【主な機能】

■ シンプルな傘判断
アプリを開くだけで、傘が「必要」「念のため持っていく」「不要」を大きなアイコンと色で瞬時にお知らせ。忙しい朝でも一目でわかります。

■ 出発地・目的地の両方をチェック
自宅と職場、自宅と学校など、出発地と目的地の両方の天気を確認。どちらか一方でも雨の可能性があれば、傘を持っていくようお知らせします。

■ GPS対応・手動選択も可能
現在地からGPSで自動的に天気を取得。位置情報を使いたくない場合は、住所や地名で検索して手動設定できます。

■ 外出時間帯を設定
「9時〜18時」「7時〜22時」など、あなたの外出予定時間を設定。その時間帯の天気だけをピンポイントで確認できます。

■ カスタマイズ可能な判断基準
「降水確率50%以上で傘必要」「降水量1mm以上で傘必要」など、自分好みの基準に設定可能。敏感な方は低めに、大丈夫な方は高めに調整できます。

■ 毎朝の通知機能
設定した時刻に傘の要否をプッシュ通知。毎朝アプリを開かなくても、通知で確認できます。

【データについて】
- 気象庁が提供する天気予報データを使用
- 15分ごとに最新データを取得
- 日本国内全都道府県に対応

【プライバシー】
- 位置情報は天気取得のみに使用
- 外部サーバーへのデータ送信なし
- すべての設定は端末内に保存

【こんな方におすすめ】
- 毎朝、傘を持っていくか悩む方
- 通勤・通学で天気が気になる方
- 出張や旅行で目的地の天気を知りたい方
- シンプルで使いやすい天気アプリをお探しの方

傘の判断は、あなたにお任せ。でも、その判断材料はこのアプリがお届けします。
```

## 審査メモ (App Review Information)

```
【アプリの概要】
このアプリは、気象庁の天気予報データを使用して、ユーザーの外出時間帯に傘が必要かどうかを判断するアプリです。

【テスト方法】
1. アプリを起動すると、位置情報の許可を求めます
2. 許可すると、現在地の天気予報を取得し、傘の要否を表示します
3. 位置情報を許可しない場合は、設定画面から手動で地点を追加できます

【位置情報の使用について】
- 位置情報は、ユーザーの現在地の都道府県を特定するために使用します
- 都道府県コードを気象庁のAPIに送信して天気予報を取得します
- 位置情報は端末内のみで処理され、外部サーバーには送信されません

【使用しているAPI】
- 気象庁天気予報API (https://www.jma.go.jp/bosai/forecast/)
  - 公開されている天気予報データを取得
- OpenStreetMap Nominatim API (https://nominatim.openstreetmap.org/)
  - 住所・地名から座標を取得（地点検索機能用）

【通知機能】
- ユーザーが設定した時刻に、傘の要否をローカル通知で送信します
- 通知はオプションで、デフォルトはオフです

【デモアカウント】
不要（ログイン機能なし）
```
