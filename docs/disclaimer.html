重要: 本アプリの情報を参考にする際は、必ず実際の天気予報や空模様もご確認ください。

1. 天気予報データについて

本アプリで使用している天気予報データは、気象庁が提供する公開データを利用しています。データの正確性については気象庁の責任範囲となり、本アプリの運営者は保証いたしません。

気象庁のデータ更新タイミングにより、最新の予報と異なる場合があります。

2. 傘判断ロジックについて

本アプリの傘判断は以下の基準に基づいています：

・降水量1mm以上 → 傘が必要
・降水確率50%以上 → 傘が必要
・降水確率30%以上 → 念のため傘を推奨
・それ以外 → 傘は不要

この判断はあくまで参考情報であり、実際の天候状況を保証するものではありません。

3. 位置情報について

GPS機能を使用した位置情報は、端末の精度や電波状況により誤差が生じる場合があります。正確な地点の天気予報が必要な場合は、設定画面から手動で地域を選択してください。

4. 通知機能について

プッシュ通知は、端末の設定やバッテリー状態により、指定時刻に届かない場合があります。重要な予定がある場合は、アプリを直接開いて確認することをお勧めします。

5. サービスの継続性

本アプリは予告なくサービスを変更、停止、または終了する場合があります。また、気象庁のAPIの仕様変更により、一時的にサービスが利用できなくなる可能性があります。

6. 損害の免責

本アプリの利用により生じた直接的、間接的な損害について、運営者は一切の責任を負いません。これには以下が含まれますが、これらに限定されません：

・傘を持参しなかったことによる濡れ被害
・傘を持参したが雨が降らなかった場合の不便
・通知の遅延や未達による影響
・その他、本アプリの使用に関連するあらゆる損害

7. データの保存について

本アプリはすべてのデータを端末内にのみ保存します。アプリの削除やデータクリアにより、設定情報が失われることがあります。
